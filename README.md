# Servo Motors Sweep Task
------------------------------------------------------------------------------------------------
## Description

This project controls four servo motors using Arduino in Tinkercad.
------------------------------------------------------------------------------------------------
### Task
- Run all four servo motors using the Sweep example for 2 seconds.
- After 2 seconds, all servo motors stop and hold at 90°.
